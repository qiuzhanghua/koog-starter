# Learn Koog

Welcome to the Koog project! This repository contains resources and examples to help you get started with Koog, a powerful framework for building applications in Kotlin.


## Prerequisites
- Java Development Kit 21 (LTS)
- Kotlin 2.2 or higher
- Gradle 9.2 or higher

- DeepSeek API Key (sign up at [DeepSeek](https://platform.deepseek.com/))


## Getting Started
0. export your DeepSeek API key as an environment variable:
   ```bash
   export DEEPSEEK_API_KEY="your_deepseek_api_key"
   ```

1. build/run the project using Gradle:
   ```bash
   gradle build
   gradle run
   ```
